# Phase 21: Личные карточки учеников

## Описание фазы
Реализация личных карточек учеников: профиль ученика, история всех уроков с результатами, статистика (баллы, средний балл, количество кирпичиков, badges), график прогресса, разбивка баллов по категориям.

## Зависимости
Phase 15: Система баллов и кирпичиков, Phase 16: Управление учениками (Pupils)

## Оценка времени
6-8 часов
